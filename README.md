# Student_Portal
## Overview
Student_Portal is a web app for students to manage profiles, view grades, and access course materials.
## Features
- Student profile management
- View and download grades
- Access and download course materials
- Secure authentication
- Responsive user interface
- Notifications for updates and deadlinesg
- Instructor and admin dashboards
- Assignment submission and tracking
- Course enrollment management
- Support for multiple user roles
